# Calorify-app
Calorify es una aplicación web de seguimiento de calorías diseñada para ayudar a los usuarios a monitorear su ingesta diaria de alimentos en relación con sus objetivos nutricionales personalizados. La aplicación permite a los usuarios registrar las calorías consumidas a lo largo del día y les proporciona una visualización en tiempo real de su progreso hacia una meta calórica preestablecida.
El proyecto está construido como una aplicación de página única (SPA) utilizando React y optimizada con Tailwind CSS para un diseño completamente responsivo y moderno. La persistencia de datos y la autenticación de usuarios se manejan mediante Google Firestore y Firebase Auth, cumpliendo con el requerimiento de almacenamiento persistente, pero utilizando una solución en la nube (Firestore) en lugar de local (localStorage/IndexedDB).
